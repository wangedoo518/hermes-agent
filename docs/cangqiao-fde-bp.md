## 1. 一句话定位

**仓桥智能用「AI Agent（Hermes）+ 小红书运营」的真实商业项目当训练场，把大三大四学生在 90 天内培养成 AI 落地交付工程师（FDE），一边用学生 + AI 低成本交付中小商家的小红书运营需求（项目仓自我造血），一边把学生送进大厂或带他们接单变现（人才仓出口）。**

我们不是培训公司，也不是代运营公司，而是**用真实交付项目训练人才、再用人才交付更多项目的「FDE 人才工厂」**。

两个资产池，一座桥：

- **项目仓**：中小商家的小红书运营需求 + AI Agent 交付系统 + 行业 SkillPack + 真实交付案例与数据。这是**会产生现金流**的训练场。
- **人才仓**：大三大四学生，经 V1-V7 体系训练成 AI FDE，按等级承接项目仓任务，最终进大厂或独立接单。
- **桥**：把「学生缺真实项目」与「商家缺低价交付」两个痛点对接，AI 做重复劳动，人做审美/沟通/终审。

## 2. 为什么是现在、为什么是我们能做（投资亮点）

1. **踩中 2025-26 增速最快、最不饱和的 AI 岗位**：FDE（Forward Deployed Engineer / AI 落地交付工程师）招聘量 YoY **+1165%**，OpenAI 2026 年 5 月成立 The Deployment Company（融资 $4B+）并收购 150 人 FDE 团队，Anthropic 成立 $1.5B 企业 JV。中国大模型应用人才缺口 **300-400 万**，雇主反复抱怨「不缺会调模型的，缺能把 AI 真正用起来产生价值的人」。

2. **训练场自带现金流，不烧钱养人**：传统职业培训是纯成本中心；我们的训练场是**小红书代运营业务本身**——3.5 亿 MAU、商家数 2024 年同比增 **8 倍**、中小商家数同比增 **379%**，需求洪流真实存在。学生边学边交付边赚钱，项目仓收入对冲培训成本。

3. **三套已验证系统，不是从零起步**（详见 §6）：
   - **路飞设计求职辅导**已跑通知识付费 IP 全链路（167 单成交、155 学员、369 次会议）；
   - **刘一欣求职顾问 Copilot**（career-coach-copilot）已是生产级后台（5 角色流水线、17 工具、405 测试全绿、L1-L4 隐私边界、云端常驻）；
   - **茶文化小红书项目**验证了「AI 高效内容 + 人做情绪价值」的本地生活样板。

4. **定位天然规避行业最大法律风险**：现有职业培训靠「保 offer / 包就业」获客，是黑猫投诉和培训贷的重灾区（求职培训投诉中培训费+培训贷占比 **54.73%**）。我们用「**陪跑不包 offer + 双出口（入职 OR 接单）**」结构，把第三方录取这个不可控变量从承诺里剔除，把行业最大负债变成最稳护城河。

5. **社会价值叙事完整**：一边帮中小商家低成本用上 AI，一边帮学生拿到真实项目经验和收入，天然适配高校就业、园区、地方 AI 产业生态——可对接政策红利（2026 被定为国家「AI 教育元年」）。

## 3. 核心打法：90 天跑通「一个大三学生从 0 到拿下 FDE 岗」的闭环

我们不追求一上来就铺大盘子。**全公司前 90 天只做一件事：把单个学生的完整闭环跑通并录下来。** 这个闭环是一鱼三吃：

```
招募 1 个大三学生（0 基础）
  ↓ V1-V7 训练（Python/SQL → LLM应用 → RAG → Agent → 真实交付项目）
  ↓ 接 1-2 个真实小红书商家订单（AI Agent 辅助交付，公司派单）
  ↓ 产出：① 商家账号真实数据增长 ② 学生的可展示交付作品集
  ↓ 出口 A：用作品集冲大厂 FDE 面试/内推
  ↓ 出口 B：独立接单变现（小红书代运营/AI 工具开发）
  ↓
录成案例视频 →
  ① 个人 IP 内容（小红书 vol.X 系列招生引流）
  ② 训练营招生案例（证明「0→交付岗」可复制）
  ③ 项目仓的第一块拼图（学生成为可派单的 V3 交付人力）
```

> **这个闭环为什么是整个商业模式的「原子」**：它同时验证了供给（学生能被训出来）、需求（商家愿付费）、出口（学生能变现）、获客（案例能招生）四件事。跑通 1 个，就能复制 N 个；这也是路演时最硬的弹药——投资人看的不是 PPT，是一个真实学生从 0 到交付的录像。

## 4. 市场论证（双边都已验证）

### 4.1 需求侧 A——FDE 岗位（人才出口）

- **岗位爆发**：FDE 招聘量 YoY +1165%；2026 初统计 39 家 AI 公司开放 224 个 FDE 岗；58% 的 FDE 岗在 11-200 人成长期创业公司（接单/低门槛入职的现实基础）。
- **中国对应岗**：解决方案工程师 / 交付工程师 / AI 落地工程师 / 大模型应用开发工程师 / AI 解决方案架构师。字节 2026 校招 5000+ 人（R&D +23%）、阿里 7000+ offer（AI 岗 >60%）。
- **薪资**：大模型应用开发 ~¥37,336/月；AI 解决方案架构师 top 年薪 ~¥150 万；脉脉数据 **>50% 的 AI 应届岗月薪 >¥5 万**；入门 AI 运营 ~¥18,457/月（陪跑入门档地板）。
- **缺口真实**：300-400 万人才缺口，63% 企业争抢大模型应用人才；MIT NANDA 2025 报告 **95% 企业 AI 试点没有可衡量业务影响**——部署鸿沟是结构性的，FDE 溢价预计持续到 2026-27。

### 4.2 需求侧 B——学生（人才入口）

- **2026 届高校毕业生预计 1,270 万人**（创历史新高，+48 万 YoY），就业竞争白热化，付费意愿强。
- **通杀筛选门槛 = 项目经验/实战**：「普通简历过不了初筛」「卡在人才库没有面试」。**我们的楔子：毕业交付项目本身就是那个缺失的简历资产。**
- 现有培训卖「技能无证据」（训练营）或「证据无技能」（求职咨询），**没有人提供「真实交付作品集」**——而这恰是大厂筛选和接单市场都认的硬通货。

### 4.3 供给侧——小红书运营项目（训练场 + 现金流）

- **平台底盘**：MAU 约 3.5 亿，日均搜索近 6 亿次，2025 电商 GMV 破 8500 亿。
- **需求洪流**：2024 年商家数同比增 **8 倍**；2024 上半年中小商家数同比增 **379%**、中小商家 GMV 增 **436%**——海量增量 SMB 涌入但缺运营能力，被平台推着必须做内容。
- **价格带空白（我们的切入点）**：传统代运营受制于运营岗 ~¥33,000/月 的人力成本，月费被迫报到 2-3 万才有毛利，**系统性放弃了 3,000-8,000 元/月 的海量 SMB 需求**。这个被传统机构嫌利润薄的价格带，正是 AI+学生模式的最佳战场。
- **天然复购**：账号养成需 3-6 个月见效，代运营合同普遍 3 个月起——天然产生续约。

### 4.4 接单兜底（出口 B 的可信度）

学生即使不进大厂也能变现，双出口结构成立：

- AI 辅助下单项目交付周期从 7-10 天压到 2-3 天，可并行多单、月收入翻倍。
- 实证案例：小程序单项 ¥2,200/5 晚；小红书提示词外包 ¥800/套；小红书代运营学生兼职月入 ¥2,000-3,000；AI 副业新手月入 ¥3,000-5,000 起步（注：均为公开自述案例，存在幸存者偏差，BP 中按「区间」而非中位数呈现）。
- 平台：闲鱼 + 小红书（最低门槛，学生实际起步地）、猪八戒、程序员客栈、知乎接单广场。

## 5. 为什么「AI Agent（Hermes）+ 小红书运营」是最完美的 FDE 训练场

这是本 BP 最核心的洞察，也是我们区别于所有竞品的根。FDE 的能力 = 拆需求 + 写代码/搭系统 + 客户沟通 + 把 AI 落地到生产并对结果负责。小红书运营 + AI Agent 交付**恰好把这套能力的每一环都变成了可练习、可计件、可验收的真实任务**：

| FDE 核心能力 | 小红书 + AI Agent 项目里的对应练习 |
| --- | --- |
| **拆需求**（把模糊业务问题变成可交付 spec） | 接到一个茶饮/美妆/本地店商家，把「我想做小红书」拆成选题策略、爆款结构、交付节奏、验收指标 |
| **搭 AI 系统**（RAG/Agent/自动化落地） | 用 Hermes 搭对标采集 Agent、爆款拆解 Agent、文案生成、评论情报、数据复盘——**这就是一套真实的 AI 落地交付系统** |
| **客户沟通**（向非技术客户翻译价值） | 跟商家对齐审美、解释数据、管理预期——FDE 50-75% 时间在做的事 |
| **对生产结果负责** | 笔记的赞藏/曝光/涨粉/转化即时回流，系统跑不跑得通一目了然——和 FDE「系统在生产环境跑起来」的成功指标同构 |
| **产出可交付证据** | 真实账号增长数据 + 交付物 = 简历项目经验 + 接单作品集 |

**三重红利同时发生：**

1. **学生在「真实交付 AI 系统」中学会 FDE**——不是上课听概念，是亲手把 AI Agent 部署到客户业务里跑出结果。这正是大厂 FDE 面试要的「你落地过吗」。
2. **AI Agent 把 60-70% 重复劳动吃掉**（选题/拆解/文案/排版/复盘已被 AI+RPA 证明可自动化：1 小时 10 篇、单篇 4 小时、CPM 低 40%），所以一个**学生在 AI 加持下能并行交付多个账号**——证明学生具备「用 AI 放大交付能力」这一 FDE 的本质能力。
3. **训练场自我造血**——商家月费覆盖训练成本，毛利结构远优于纯人工机构（把传统需 4-8 万/月团队成本的服务，压到 3,000-8,000 元/月价位仍能盈利）。

> 一句话：**别人的训练场是模拟题，我们的训练场是真实战场，而且这个战场自己赚钱。** 学生学的不是「小红书运营」，是「如何用 AI Agent 把一个非标业务需求低成本落地交付」——把垂类换成企业客服、电商、求职咨询，能力直接迁移。小红书只是第一个最易起量、项目体量最大的练兵赛道。

## 6. 已验证资产：三个项目仓案例

### 6.1 路飞设计类求职辅导（知识付费 IP 全链路样板）

- **验证了什么**：一个垂类 IP 从内容（vol.X 沉浸式模拟面试，爆款点赞 2802/收藏 2700）→ 公域→私域漏斗（双微信入口路由）→ 1V1/课程成交（¥29.9 引流 → ¥99 工作坊 → ¥280 1V1）的完整变现。已沉淀 167 单、155 学员、369 次会议、爆款拆解十维方法论、CES 算法。
- **复用到 FDE 教练**：换垂类（设计求职 → FDE 求职），整套人设运营机器 + 内容工作室（11 个 Skill + 3 个 MCP 工具）直接迁移。

### 6.2 刘一欣求职顾问 Copilot（交付后台样板）

- **验证了什么**：求职顾问的后台 AI 副驾驶——5 角色流水线（Elon 意图分流 → Jobs 服务诊断 → Linus 草稿生成 → Turing 6 维质检 → Bezos CRM 扫学员）、17 个独立工具、405 测试全绿、L1-L4 隐私边界、HITL 红线（AI 只出草稿，顾问审核后本人发出）、云端常驻 + 手机半自动转发。
- **复用到 FDE 工厂**：直接 fork 成 `fde-coach-profile`（2-3 周改垂类），把 Jobs/Linus 的知识库换成 FDE 内容，profile-builder 改造成「FDE 候选人画像（接 GitHub 项目）」。这套同样是学生要学着搭的「AI 落地交付系统」教学样板。

### 6.3 茶文化小红书项目（本地生活交付样板）

- **验证了什么**：「AI 负责高效率内容/数据/分析，人负责审美/信任/成交」的人机协同模型；茶文化的情绪价值降低 AI 接受门槛；杭州生活馆作为线下流量入口和 AI 体验空间。
- **复用到项目仓**：本地生活/茶饮是小红书代运营的高频高复购赛道，直接成为学生的第一批真实交付订单来源，也是线下讲座/路演的实景场地。

> 三个案例共同证明：我们已经有**内容运营机器（路飞）+ AI 交付后台（刘一欣）+ 真实客户场景（茶文化）**，缺的只是把它们串成「学生训练—项目交付—人才出口」的飞轮，并跑通第一个闭环。

## 8. 竞争格局与护城河

| 类型 | 代表 | 价格 | 缺口 | 我们的差异化 |
| --- | --- | --- | --- | --- |
| 通用 AI 训练营 | 黑马/传智、千锋 | ¥20,980-27,980 脱产 | 课纲驱动、无真实客户作品集、「包就业」负债、大班质量稀释 | 真实交付项目产出作品集，小班陪跑 |
| 求职咨询/陪跑 | 职优你、Offer 帮 | ¥6,888-39,800 | 只做简历/内推/面试，0 技术能力沉淀，退款纠纷、留学生为主 | 既建技术能力又建作品集，双出口 |
| 通用职业教练 | 张雪峰/峰学蔚来 | 梦想卡 ¥8,999-18,999 | 做志愿填报，不做求职交付 | 聚焦 FDE 求职交付 |
| 小红书代运营机构 | 各代运营公司 | ¥2-3 万/月 | 人力成本高，放弃 SMB 价格带 | AI+学生切 ¥3,000-8,000 空白带 |

**核心护城河（一句话）：竞品要么卖「技能无证据」（训练营），要么卖「证据无技能」（求职咨询），且都背着「包 offer」退款风险；我们卖「已交付的 AI 落地作品集 + 分发能力」，且通过两个独立出口变现——从而消除了定义这个行业投诉模式的退款负债。**

四道叠加的壁垒：

1. **真实项目数据持续沉淀**（项目仓越大，案例越多，教学越强，派单越多）。
2. **AI 交付系统工程化能力**（Hermes + Profile + SkillPack + 17 工具 + 质量门 + HITL）。
3. **项目仓 ↔ 人才仓供需飞轮**（学生既是付费方又是交付方，双边自增强）。
4. **FDE 定位的先发占位**（没人占住「AI 落地交付」这个最高 ROI、最不饱和的细分）。

## 9. 单元经济与可行性测算（路演口径，需用真实闭环校准）

**学生侧（人才仓）单经济示例：**
- 一期 V1-V7 训练营客单 ¥4,980，20 人/期 → 收入 ¥99,600/期。
- 引流漏斗对标：9.9 引流课 + 企微绑定好友添加率 95-100%；私域 ¥9.9→¥2,980→¥10k+ 阶梯。
- 高客单 6 个月全陪跑 ¥15,000 × 10 人 → ¥150,000。

**商家侧（项目仓）单经济示例：**
- 1 个学生在 AI 加持下并行交付 3-5 个账号 × ¥5,000/月 = ¥15,000-25,000/月营收。
- 学生分成 + AI/工具成本 < 传统 ¥33,000/月 全职运营，毛利显著为正。
- 学生既贡献项目仓营收，又在过程中完成 FDE 训练——**同一笔投入，两端产出**。

**毛利提升逻辑：**
- 每交付一个项目沉淀一套模板，下一次交付成本下降。
- 人才等级化后低复杂度任务由 V1/V2 完成，核心团队聚焦架构与验收。
- 行业 SkillPack 复用后平台订阅/佣金占比提升。

## 10. 90 天里程碑（PMV 验证期）

### 第 0-30 天：人设 + 后台 + 第一批内容

- 谈下 2-3 个茶饮/本地生活商家作为首批项目仓订单（可用茶文化项目场地切入）。

### 第 30-60 天：第一个学生闭环
- 招募 3-5 个种子学生，免费/低价跑 V1-V5。
- 学生在 AI Agent 辅助下交付第 1-2 个真实商家账号，产出**可展示的交付作品集 + 真实账号数据**。
- 办第一场高校线下讲座（浙大/浙财），现场录制成 vol.X 素材。
- 私域群起量（目标 100-200 人）。

### 第 60-90 天：闭环录制 + 首期成交
- 把「0 基础学生 → 毕业交付项目 → 面试/接单」录成案例视频（路演核心弹药）。
- 跑通 ≥1 个学生「大厂 FDE 面试」或「独立接单成交」实证。
- 开第一期付费 V1-V7 训练营（¥2,980-6,980），目标 10-20 人。
- 项目仓达到 5-8 个付费商家账号，验证造血。

**90 天关键结果（融资证据）：** 1 个完整闭环录像 + 5-8 个付费商家 + 10-20 个付费学生 + 第一个面试/接单实证。

## 13. 风险与应对（含尽调诚实披露）

| 风险 | 应对 |
| --- | --- |
| 过度承诺「保进大厂」 | **红线禁止**；只承诺过程+作品集+派单机会；接单兜底消解二元退款纠纷 |
| 第一个闭环跑不通，案例空心 | 全公司前 90 天只聚焦 1 个闭环；跑不出真实交付+实证就不开高客单 |
| 学生交付质量不稳定 | V1-V7 分级 + Turing 6 维质检 + 核心团队终审 + HITL |
| 小红书平台风控/封号 | 慢速采集、授权/公开数据、人工转发闭环、多账号矩阵分散 |
| 9.9 引流课已饱和、获客成本上升 | 不靠纯价格阶梯，靠「真实闭环案例」差异化获客；高校讲座做低成本入口 |
| FDE 知识库自身覆盖不全 | 用三个真实项目反向沉淀；不会的领域请外部 FDE 客座，不硬编 |
| **数据可信度（尽调会被问）** | 小红书报价、AI 培训 TAM（¥8000 亿）、就业率（93.5%）等多为机构自发布或二手数据，融资前抽样复核 5-10 家真实报价与 3-5 个学员真实出口，BP 中以区间+「公开口径」标注 |
| 学生数据隐私 | 照搬 career-coach 的 L3/L4 隔离 + HITL，学员/客户数据永不进 git、永不自动外发 |
